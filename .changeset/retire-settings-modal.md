Removed the settings modal (the toolbar's gear icon) — submit button label and mode now live in the sidebar's "Design" tab alongside theme, spacing, and button-style controls, so all form-level configuration is in one place. That submit section, and the choice between combined/per-section submit buttons, is now hidden entirely for documents that have no fillable fields (e.g. pure display/landing pages) or where only one section actually has fields.
